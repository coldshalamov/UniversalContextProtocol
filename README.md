# Universal Context Protocol (UCP)

## Overview
A dynamic context management system that solves "Tool Overload" by injecting relevant tool schemas based on conversation context.

## Structure
- **research/**: Reference papers and external codebases.
- **reports/**: Audits and status reports.
- **docs/**: Design plans and synthesis documents.
- **src/**: Source code (Python).
- **tests/**: Unit and integration tests.
